

# YouTube Channel Branding Package for 1325AI

## Overview
Create professional YouTube channel assets that maintain brand consistency with Kayla's neural brain logo design.

---

## Assets to Generate

### 1. YouTube Banner Image (2048 x 1152 pixels)
**Design concept:**
- Deep navy blue gradient background matching the logo
- Neural network pattern extending horizontally across the banner
- Gold glowing nodes scattered throughout for visual interest
- Logo positioned on the left or center-left
- Tagline "Building the Future of Digital Commerce | AI" on the right
- Subtle vignette effect to draw focus to the center

**Safe zone considerations:**
YouTube displays different portions of the banner on different devices:
- Desktop: Full 2560 x 423 px visible
- Tablet: 1855 x 423 px visible  
- Mobile: 1546 x 423 px visible
- TV: 2560 x 1440 px visible

The key content (logo + tagline) will be centered in the "safe area" (1546 x 423 px).

### 2. Video Watermark (150 x 150 pixels)
**Design concept:**
- Just the neural brain icon (no text)
- Transparent background
- Semi-transparent gold/white for subtlety
- Clean and recognizable at small sizes

---

## Implementation Steps

### Step 1: Generate Banner Image
Use AI image generation to create a professional YouTube banner:
- Aspect ratio: 16:9 landscape (2048 x 1152)
- Style: Match the existing logo aesthetic
- Elements: Neural network, gold accents, deep blue background

### Step 2: Generate Watermark
Create a simplified icon version:
- Square format (150 x 150)
- Transparent PNG
- Brain network only, no text

### Step 3: Store Assets
Save generated images to the project for download:
- `public/youtube/youtube-banner.png`
- `public/youtube/youtube-watermark.png`
